== Description ==
Easy Event Creator plugin allows you to create and display events on your website with ease. You can create custom events with detailed information such as event type, date, description, and URL. Then, you can display these events on a specific page using the [events] shortcode.

The plugin provides configuration options to control what information is displayed in the events, such as title, event type, date, description, image, and URL. You can also customize the height of the event boxes to fit your design.

== Features ==
- Create custom events with detailed information.
- Display events on a page using the [events] shortcode.
- Control what information is displayed in the events (title, event type, date, description, image, URL).
- Customize the height of event boxes.
- Fully compatible with WordPress 5.8 and previous versions.

== Installation ==
1. Upload the `Easy-Event-Creator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin options on the plugin's 'Settings' page.

== Usage ==
1. Create new events by navigating to 'Events' in the admin menu.
2. Fill in the event information fields (event type, date, description, URL, etc.).
3. Insert the `[events]` shortcode on any page to display the list of events.

== Frequently Asked Questions ==
1. Can I display only events from a specific category?
   Yes, you can use the 'category' parameter in the [events] shortcode to filter events by category. For example: [events category="concerts"]

2. How can I change the style of the events?
   You can customize the appearance of the events by modifying the 'style.css' file of the plugin. This file is located in the plugin folder (/wp-content/plugins/easy-event-creator/).
